
JDBC - Java DataBase Connection
  >자바로 데이터베이스 연결 및 관련 작업을 할 때 사용
 JDBC 드라이버가 필요함

커넥션 풀
DB 작업을 위해서 웹서버 Connection 객체를 생성할 때마다
네트워크 통신이 이루어지며 사용자 인증 같은 시간이 걸리는 작업을 진행함.
자주 커넥션과 해제는 시스템 성능에 영향을 미친다.
이를 해결하기 위해 커넥션 풀을 사용함

커넥션 풀은 미리 Connection 객체를 생성해 두고 요청이 있을 때 이미 생성된
Connection 객체를 가져다가 사용하는 방법.

직접 만들어서 사용하지 않고 WAS가 제공하는 것을 이용하는 게 좋다.

(과정 설명)
1.WAS(톰캣) 가 시작할 때 server.xml, context.xml에서 설정한 대로 미리 커넥션풀을 생성함.
2.JSP 코드에서 JNDI 서버로부터 데이터소스 객체를 얻어온다.
3.데이터소스에서 커넥션 객체를 가져온다.
4.DB 작업 수행
5.작업이 끝나면 커넥션 객체를 풀에 반환함

http://localhost:80/
